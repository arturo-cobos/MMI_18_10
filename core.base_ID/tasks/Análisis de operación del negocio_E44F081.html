Tarea: Analizar operación del negocio
Busca definir la estrategia de transformación de una descripción del comportamiento a nivel del sistema de negocio, en la estructura del sistema de a alto nivel.
Disciplinas: Análisis de Negocio e Ingeniería de Requerimientos
Objetivo
  • Identificar variables críticas que exigen mayor control
  • Reducir riesgos del proyecto
  • Evaluar el estado actual de la operación
  • Conocer el entorno en el cual se va a desarrollar el proyecto
  • Identificar Stakeholders y responsables de las actividades 
  • Identificar fuentes de requisitos
