Tarea: PRB_T4_Estimación etapa de requerimientos
Estimar el tiempo y esfuerzo requerido para las actividades de ingeniería de requerimientos, asociados con la solicitud.

Para solicitudes clasificadas como de gran tamaño, se configura y estima el proceso de ingeniería de requerimientos para una primera fase, y para las siguientes por estimación progresiva.
Disciplinas: PRB
Relaciones
RolesRealizador principal:
  • Ingeniero de requerimientos
Otras actividades adicionales que realiza:
EntradasObligatoria:
  • FSP cerrado
  • Inventario de Fuentes
    		Opcional:
    • Ninguno
    Salidas
    • Esimación de tiempo y esfuerzo
    • Matriz de configuración de proceso
      Pasos
      Analizar el contenido de la Solicitud.
      Definir las Actividades.
      Secuenciar las Actividades.
      Estimar los recursos.
      Estimar el esfuerzo requerido.
      Estimar la duración.
      Elaborar matriz de configuración.
