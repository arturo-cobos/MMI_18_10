Tarea: IIBA_Planear comunicaciones
El plan de comunicaciones describe la estructura propuesta y el calendario para las comunicaciones respecto a las actividades de análisis de negocio. Graba y organiza las actividades para proveer una base para establecer las expectativas para el trabajo de análisis de negocio, reuniones, revisiones y otras comunicaciones
Disciplinas: Análisis de Negocio e Ingeniería de Requerimientos
Objetivo
  • Establece la metodología para las comunicaciones dentro del proceso de análisis de negocio y levantamiento de requerimientos
  • Determina el calendario para las comunicaciones
  • Permite la planeación de reuniones, revisiones y demas metodos para transmitir información
Relaciones
RolesRealizador principal:
  • Ingeniero de Requerimientos
